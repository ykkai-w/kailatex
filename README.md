# kailatex-skills

把碎片化的资料整理成工整、清晰排版的 PDF 的一套规范与脚本。

现在的信息大多是碎片化攒下来的：AI 帮你总结的要点、随手记的笔记、拍下来的截图、东拼西凑的题库。内容是有了，可它们散落各处、格式各异，真要拿来用、打印或归档，仍然得花不少功夫重新整理。kailatex 把这道工序固定下来：一套排版规范，配上转换和复查脚本，把零散的输入整理成版式统一、可直接打印的文档。规范本身写成 skill，方便交给 AI 复现，每次产出都守同一套标准，而不是每回重新发挥。

## 适用

- 把 AI 输出的碎片总结、笔记、截图、题库等整理成工整文档
- 以中文为主、含公式与表格的资料排版（XeLaTeX 加 ctex）
- 需要固定版式的资料：题目在前答案紧随、目录快速定位、列点序号原样保留等
- 考试复习资料只是其中一种用途，并不限于此

## 组成

- `SKILL.md`：全流程与排版规范，含工具链、五步流水线、LaTeX 模板、转换器要点、风格约定、常见问题与修复，可直接当 skill 使用。
- `template_convert.py`：转换器骨架，把结构化数据转成 `.tex`，已处理数学保护、特殊符转义、中文与货币符号、表格、多行公式、圈码列点。
- `preview.py`：复查工具，用 PyMuPDF 把 PDF 转成图，整册拼图看结构，单页放大或切半核验公式与表格。

## 工作流

源（txt 或截图）到取文本（正则解析或放大后 OCR），再到结构化 JSON 并核对条目，转换器出 `.tex`，编译，转图逐页复查，最后清理与备份。

## 依赖

XeLaTeX（tectonic 便携版，自带 Fandol 中文字体）、Python（Pillow、PyMuPDF）。详见 `SKILL.md`。
